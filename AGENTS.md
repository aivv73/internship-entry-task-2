# Agent guidance

## Agent skills

### Issue tracker

Issues and PRDs live in GitHub Issues and are managed with `gh`. See `docs/agents/issue-tracker.md`.

### Triage labels

Use the canonical triage roles and matching GitHub label names. See `docs/agents/triage-labels.md`.

### Domain docs

This is a single-context repository using a root `CONTEXT.md` and system-wide ADRs in `docs/adr/` when those documents are needed. See `docs/agents/domain.md`.
